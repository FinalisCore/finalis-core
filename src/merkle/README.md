# Merkle Component

## Purpose

`src/merkle/` contains Merkle tree structures, hashing composition, and proof support used by state and protocol paths.

## Responsibilities

- Merkle root/proof computation helpers.
- Deterministic tree handling utilities.

## Non-Goals

- Consensus orchestration logic.
- Network transport details.

## Dependency Notes

- Depends on crypto and common primitives.
- Should remain a focused utility component.
