# License Policy

This document defines how license information is represented in this repository.

## Current Policy

- The authoritative license is the root `LICENSE` file.
- New source files should include an SPDX license identifier header for clarity and tooling compatibility.

Recommended header for new files:

- `SPDX-License-Identifier: MIT`

If the project license changes in the future, update this document and apply the matching SPDX identifier.

## Scope

This policy applies to source files, headers, and scripts.

## Why

- Reduces ambiguity for contributors and integrators.
- Improves compatibility with automated compliance tooling.
- Addresses inconsistent expectations around per-file license headers.
